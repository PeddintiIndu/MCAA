About os lab

An Operating System (OS) lab provides practical experience in understanding and implementing core OS concepts. 
Students typically work with Linux-based systems to explore process management, memory management, file systems, and 
other essential OS components. The lab aims to bridge the gap between theoretical knowledge from lectures and practical 
application through hands-on exercises and experiments.
